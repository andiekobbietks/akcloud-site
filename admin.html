Odoo Backend Connection
Netlify Deployment Notes
1. Enable CORS on Odoo in odoo.conf:
cors_headers = *
Or use a reverse proxy with CORS headers.

2. Odoo must be HTTPS — mixed content blocked.
3. For bookings: install appointment module.
Booking URL: {odoo_url}/web#action=appointment

4. Credentials stored in localStorage — per-device. Re-enter on new browsers.
